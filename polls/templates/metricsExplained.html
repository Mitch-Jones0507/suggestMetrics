Classification

Classification Explained:

Classification in Machine Learning is a supervised learning technique that is used to categorise data using pre-defined classes or labels. The model is trained using a dataset with known results, learning patterns/relationships between the input features and corresponding labels. Once trained, the model can then predict the class/label of the new data. Common examples of this include spam detection, image recognition and medical diagnosis. Algorithms like logistic regression, decision trees and neural networks are often used for classification tasks, but in our program we will be using both accuracy and F1-Score. Regression

Regression Explained:

Regression in machine learning is a supervised machine learning technique that is used to predict a continuous numerical value based on input features. Unlike the aforementioned classification, which predicts categories, regression estimates quantities - such as prices, sales figures of even temperatures, for example. The model learns the relationships between input variables and a continuous output from labelled training data. Common examples of this include house price predictions, stock values of energy consumption. Popular algorithms used in regression include linear regression, decision trees and support vector regression. Regression is widely used in forecasting, trend analysis, and resource planning, helping businesses and researchers make informed, data-driven decisions based on quantitative predictions.
