Classification Network

The classification network details the algorithm that the suggestMetrics system follows when the user selects a classification task. Classification refers to the general concept of measuring the agreement between predicted class labels and actual class labels.

Model Relation Network

The model relation network details the algorithm that the suggestMetrics system follows when the user selects a model relation task. Model relation refers to the general concept of measuring the alignment between predictions and actual outcomes.
