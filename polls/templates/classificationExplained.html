[C1] This question defines the target space of the task, allowing the system to analyse exactly the classes that the user is trying to predict using their model.

[C2] The system checks for the total number of classes in the user's task. This allows the system to differentiate between a binary and multiclass scenario, which possess separate features to analyse.

[C3] This question defines the positive class of the binary classification task. A positive class refers to a class that the user is actively trying to predict.

[C4] This question queries the user on their concern for particular class performance or overall model performance. Accuracy is a measure of the overall proportion of correct guesses across all classes, whereas macroF1 gives equal weight to the F1 score of each class individually. If the answer is no, accuracy is a suggested metric. If the answer is yes, macroF1 is a suggested metric. Juri Optiz, 2024

[C5] This feature analyses the number of instances (empirical class distribution) of the majority class divided by the number of instances of the minority class. If this ratio exceeds a threshold of 3 in favour of the negative class, macroF1 is suggested, whereas if this threshold exceeds 3 in favour of the positive class, accuracy is suggested. Azal Ahmad Khan, 2024

[C6] This feature analyses the number of minority classes in your multiclass system. If this number exceeds a threshold of 3, macroF1 is suggested, else accuracy is suggested. Because m reaches a notable level, a metric that is decomposable or averaged over classes is valuable to report general performance of the model. Juri Optiz, 2024

[C7] This feature analyses the 'level of imbalance' of a multiclass system. d( ζ, e ) represents a distance operator (such as euclidian distance) between the set of empirical class distributions and the balanced distribution. d( ιm , e ) represents the distance function between the distribution of exactly m minority classes with the furthest distance from e. Overall, this feature gives an idea of how imbalanced the multiclass system is. If this metric exceeds 1 it indicates that the dataset has a high distance discrepancies compared to its worst minority classes, suggesting macroF1, else suggesting accuracy. Jonathan Ortigosa-Hernández, 2017

[C8] This question queries the user on the severity of wrong predictions. In a scenario where a misclassification is problematic (such as a rare disease), the user should want to understand the performance of their model in rare classes. In this case, macroF1 is suggested. Matthew Fox, 2022
